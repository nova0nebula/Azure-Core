# Change Log
All changes will be documented here.

## Version 1.0.0
Build 1000<br>
Released on 12 February 2025<br>
- Initial release with basic functionality.

## Version 1.1.0
Build 1001<br>
Released on 12 February 2025<br>
- Added `'ac version'` & `'ac --version'` command to see Azure Core current version.
- Added `'ac changelog'` command to see the full Azure Core changelog.

## Version 1.2.0
Build 1002<br>
Released on 13 February 2025<br>
- Added `'ac checkupdates'` command for users to check if there are any new updates and update their system on the spot.

## Version 1.2.1
Build 1003<br>
Released on 13 February 2025<br>
- Fixed exit message saying "Exiting Azure Command" instead of "Exiting Azure Core".

## Version 1.3.0
Build 1004<br>
Released on 14 February 2025<br>
- Removed automatic updating when running `'ac checkupdates'`. Users now have to pull new updates from GitHub themselves.
- Fixed previous changelog commands to say the prefix in the command.
